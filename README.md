# LearnNetworkNeuron

Интерактивные HTML/CSS/JS примеры для изучения нейронов и нейронных сетей с возрастающей сложностью. Все примеры работают прямо в браузере без установки дополнительных инструментов.

## 📖 Описание примеров

### 1. Одиночный нейрон (`01-single-neuron.html`)
Визуализация работы одного нейрона (перцептрона). Вы можете:
- Изменять входные значения x₁, x₂, x₃ ползунками
- Изменять веса w₁, w₂, w₃ и смещение (bias)
- Выбирать функцию активации: Sigmoid, ReLU, Tanh, Step
- Наблюдать пошаговое вычисление: взвешенная сумма → активация → выход
- Видеть график функции активации с текущим значением

### 2. Логические операции (`02-logic-gates.html`)
Демонстрация того, как нейрон обучается решать логические задачи. Особенности:
- Поддержка операций AND, OR, XOR
- Анимированное обучение с визуализацией изменения весов
- Визуализация границы решения в 2D пространстве
- График ошибки обучения в реальном времени
- История изменений весов w₁, w₂, bias
- Для XOR: объяснение линейной неразделимости

### 3. Простая нейронная сеть (`03-neural-network.html`)
Интерактивная многослойная нейронная сеть. Возможности:
- Настройка числа нейронов в каждом слое (входной, скрытый, выходной)
- Анимация прямого распространения сигнала
- Цветовая индикация активации нейронов (синий = положительный, красный = отрицательный вес)
- Тепловая карта весов всех слоёв
- Выбор функции активации: Sigmoid, ReLU, Tanh, Leaky ReLU

### 4. Обучение в реальном времени (`04-training-visualization.html`)
Визуализация процесса обучения нейронной сети. Включает:
- Приближение функций: синус, парабола, ступенька, пила, Гаусс
- График функции потерь в реальном времени
- Анимация ландшафта потерь с «шариком» оптимизации
- Подсветка нейронов и весов при обратном распространении ошибки
- Диаграмма градиентного потока по слоям
- Настройка learning rate, momentum, размера батча, L2-регуляризации

### 5. Распознавание рукописных цифр (`05-digit-recognition.html`)
Полный цикл: рисование → предобработка → нейросеть → предсказание. Включает:
- Холст для рисования цифр мышью или пальцем (touch-поддержка)
- Архитектура 784 → 128 → 64 → 10 (ReLU + Softmax)
- Столбчатая диаграмма уверенности для каждой цифры 0–9
- Топ-3 предсказания с процентами
- Режим обучения: добавьте свои примеры и обучите сеть
- Визуализация активаций всех слоёв

## 🧠 Теоретическая справка

### Нейрон

Нейрон — основная вычислительная единица нейронной сети, вдохновлённая биологическими нейронами мозга.

```
         w₁
x₁ ─────────┐
         w₂  │
x₂ ─────────┤──▶  z = Σ(xᵢ·wᵢ) + b  ──▶  y = f(z)
         w₃  │
x₃ ─────────┘
              b (bias)
```

**Формула**: `y = f(w₁x₁ + w₂x₂ + w₃x₃ + b)`

### Функции активации

| Функция | Формула | Диапазон | Применение |
|---------|---------|----------|------------|
| Sigmoid | `1 / (1 + e⁻ˣ)` | (0, 1) | Бинарная классификация |
| ReLU | `max(0, x)` | [0, ∞) | Скрытые слои (современные сети) |
| Tanh | `(eˣ − e⁻ˣ) / (eˣ + e⁻ˣ)` | (−1, 1) | Скрытые слои |
| Softmax | `eˣⁱ / Σeˣʲ` | (0, 1) | Многоклассовая классификация |

### Обучение и градиентный спуск

Нейронная сеть обучается методом **обратного распространения ошибки** (backpropagation) с использованием **градиентного спуска**:

1. **Прямой проход** — вычислить предсказание сети
2. **Функция потерь** — измерить ошибку: `L = (y_pred − y_true)²`
3. **Обратный проход** — вычислить градиенты `∂L/∂w` для каждого веса
4. **Обновление весов** — `w ← w − lr · ∂L/∂w`

### Многослойная нейронная сеть (MLP)

```
Входной слой    Скрытый слой    Выходной слой
    x₁ ──────────▶ h₁ ──────────▶ y₁
    x₂ ──────────▶ h₂ ──────────▶ y₂
    x₃ ──────────▶ h₃
                ▶ h₄
```

Скрытые слои позволяют сети изучать нелинейные зависимости, недоступные для одного нейрона (например, XOR).

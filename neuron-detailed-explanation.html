🧠 Подробное объяснение работы нейрона

Интерактивная визуализация с пошаговыми вычислениями

Что такое нейрон? Нейрон - это математическая функция, которая принимает несколько входов, умножает их на веса, суммирует результаты, добавляет смещение (bias) и применяет функцию активации.
y = activation(Σ(xᵢ × wᵢ) + bias)

📥 Входные данные

Входы - это данные, которые нейрон получает. Это могут быть пиксели изображения, слова в тексте, или любые другие числовые значения.

Веса определяют важность каждого входа. Положительный вес усиливает вход, отрицательный - ослабляет. Bias позволяет нейрону активироваться даже при нулевых входах.

💻 Код реализации нейрона

class Neuron { constructor(numInputs) { // Инициализация весов случайными значениями this.weights = Array(numInputs).fill(0) .map(() => Math.random() * 2 - 1); this.bias = Math.random() * 2 - 1; } // Прямое распространение forward(inputs) { let sum = this.bias; for (let i = 0; i < inputs.length; i++) { sum += inputs[i] * this.weights[i]; } return this.activate(sum); } // Функция активации (Sigmoid) activate(x) { return 1 / (1 + Math.exp(-x)); } // Обучение (градиентный спуск) train(inputs, target, learningRate) { const output = this.forward(inputs); const error = target - output; const gradient = error * output * (1 - output); // Обновление весов for (let i = 0; i < this.weights.length; i++) { this.weights[i] += learningRate * gradient * inputs[i]; } this.bias += learningRate * gradient; return Math.abs(error); } }
